# BookSafe ERP System

A comprehensive ERP system built with Node.js (Backend) and Flutter (Frontend).

## Features
- Sales & POS Management
- Inventory Tracking
- Customer Management & Debt Tracking
- Reports & Analytics
- Multi-tenant Support
- Offline Synchronization

## Tech Stack
- **Backend:** Node.js, Express, Sequelize (MySQL)
- **Frontend:** Flutter, Riverpod, Dio

## Setup
1. Clone the repository
2. Configure environment variables in `backend/.env`
3. Run `npm install` in `backend` and `npm start`
4. Run `flutter pub get` in `frontend` and `flutter run`
